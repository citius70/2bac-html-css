    6. El modelo de caja (Box Model)¶

    Todos los elementos HTML son tratados por el navegador como contenido dentro de una caja rectangulare. Todo elemento (<p>, <div>, <h1>, <img>, etc.) tiene esta estructura de caja, llamado box model (o modelo de cajas).

    Hagamos un símil. Imagina que tienes unos zapatos (tu contenido: texto, imagen, botón) que vas a enviar por correo. Para protegerlos, los metes en una caja de protección:

    6.1 Elementos del modelo de caja¶

    6.1.1 Content (contenido)¶

    El contenido es el elemento interior de la caja, como un párrafo, una imagen o un botón. En nuestro ejemplo, el contenido son los “zapatos” del box model. En el dibujo del modelo de caja, corresponde al rectángulo blanco central.

    Se dimensiona mediante las propiedades:

    width  /* ancho */
height /* alto */

    Por ejemplo:

    width: 100px;
height: 30px;

    Para visualizar mejor el tamaño del contenido, se puede usar la propiedad background-color.

    <p style="width: 200px; height: 2000px; background-color: lightblue;">
    Lorem ipsum dolor sit amet, consectetur adipiscing elit. Donec finibus consectetur nibh eget faucibus.
</p>

<p style="width: 300px; height: 100px; background-color: lightgreen;">

</p>

    6.1.2 Border¶

    El border es el borde de la caja que rodea el contenido. Si no se define, el borde es invisible por defecto.

    Las posibles propiedades son:

    Propiedad Qué hace Ejemplo Opciones disponibles / comentarios
    border-width Define el grosor del borde border-width: 5px; 1px, 2px, 3px, 5px, 10px… Puede usar unidades px, em o % (menos frecuente).
    border-style Define el tipo de línea del borde border-style: solid; none, solid, dashed, dotted, double, groove, ridge, inset, outset, hidden
    border-color Define el color del borde border-color: blue; Nombres de color (red, green), hexadecimal (#ff0000), RGB (rgb(0,0,255)), RGBA (rgba(0,0,255,0.5))
    border Atajo que combina las tres anteriores border: 5px solid blue; Equivalente a border-width + border-style + border-color

    Debido a que la caja tiene cuatro lados (border-top, border-bottom, border-lefty border-right), cada una de ellas puede configurarse individualmente. Ejemplo:

    • border-right: 3px dashed green;
    • border-bottom: 3px dotted red;

    también es válido:

    • border-bottom-color: green;

    Ejemplo: 

    <p style="width: 400px; height: 70px; border: 4px solid; border-bottom: dotted blue;">
  Lorem ipsum dolor sit amet, consectetur adipiscing elit. Donec finibus consectetur nibh eget faucibus.
</p>

    6.1.3 Padding¶

    El padding es el espacio interior de la caja, es decir, el espacio entre el contenido y el borde. En el dibujo se representa con el área amarilla.

    Las posibles propiedades son:

    Propiedad Qué hace Ejemplo Opciones disponibles / comentarios
    padding-top Define el espacio superior interno padding-top: 10px; Puede usar unidades px, em, %
    padding-bottom Define el espacio inferior interno padding-bottom: 15px;
    padding-left Define el espacio interno izquierdo padding-left: 20px;
    padding-right Define el espacio interno derecho padding-right: 5px;
    padding Atajo para definir los cuatro lados a la vez padding: 10px 20px 5px 15px; Orden: arriba, derecha, abajo, izquierda

    Ejemplos de uso:

    • padding: 20px; → aplica el mismo valor a los cuatro lados.
    • padding: 10px 30px; → aplica 10 px arriba y abajo, y 30 px a izquierda y derecha.

    Ejemplo:

    <p style="width: 80%; height: 100px; border: 3px solid blue; padding: 820px; padding-left: 90px;">
Lorem ipsum dolor sit amet, consectetur adipiscing elit. Donec finibus consectetur nibh eget faucibus.
</p>

    Lorem ipsum dolor sit amet, consectetur adipiscing elit. Donec finibus consectetur nibh eget faucibus. 

    <p style="width: 350px; border: 2px solid blue; padding: 5px; background-color: lightyellow;">
Párrafo con <strong>padding pequeño</strong>. El texto está cerca del borde.
</p>

<p style="width: 350px; border: 2px solid blue; padding: 25px; background-color: lightyellow;">
Párrafo con <strong>padding grande</strong>. El texto está más separado del borde.

    6.1.4 Margin¶

    El margin es el espacio exterior de la caja, es decir, el espacio entre el borde del elemento y otros elementos que lo rodean.
    En el dibujo se representa de azul claro.

    Las posibles propiedades son:

    Propiedad Qué hace Ejemplo Opciones disponibles / comentarios
    margin-top Define el espacio exterior superior margin-top: 10px; Puede usar unidades px, em, %
    margin-bottom Define el espacio exterior inferior margin-bottom: 15px;
    margin-left Define el espacio exterior izquierdo margin-left: 20px;
    margin-right Define el espacio exterior derecho margin-right: 5px;
    margin Atajo para definir los cuatro lados a la vez margin: 10px 20px 5px 15px; Orden: arriba, derecha, abajo, izquierda

    Ejemplos de uso:

    • margin: 20px; → aplica el mismo valor a los cuatro lados.
    • margin: 10px 30px; → aplica 10 px arriba y abajo, y 30 px a izquierda y derecha.
    • margin: 0 auto; → centra horizontalmente un elemento con ancho fijo.

    Aquí tienes un ejemplo con dos párrafos para explicar margin, mostrando claramente cómo funciona el espacio exterior entre los elementos:

    <p style="width: 350px; border: 2px solid blue; padding: 10px; margin: 5px; background-color: lightyellow;">
Párrafo con <strong>margin pequeño</strong>. El espacio entre este párrafo y el siguiente es reducido.
</p>
<p style="width: 350px; border: 2px solid blue; padding: 10px; margin: 25px; background-color: lightyellow;">
Párrafo con <strong>margin grande</strong>. El espacio entre este párrafo y el anterior es mucho mayor.
</p>

    Párrafo con margin pequeño. El espacio entre este párrafo y el siguiente es reducido.

    Párrafo con margin grande. El espacio entre este párrafo y el anterior es mucho mayor.

    6.2 Ejemplo de box model¶

    archivo.html
    <p class=tarjeta>Lorem ipsum dolor sit amet, consectetur adipiscing elit. Maecenas vitae interdum felis. Morbi varius lobortis mi sed aliquam. Donec finibus sapien sem, a scelerisque elit lacinia at.</p>
    archivo.css
    .tarjeta {
    /* El espacio para el elemento (zapatos) mide 300px  y tendrá un color rosa*/
    width: 300px;  
    background-color: pink;         

    /* 20px de espacio interior (burbujas) menos el inferior de 60px */
    padding: 20px;
    padding-bottom: 60px;          

    /* Borde azul de 2px y de 4px a puntos el borde inferior (caja de cartón) */
    border: 2px solid blue; 
    border-bottom: 4px dotted;

    /* 30px de espacio izquierdo exterior (aire) */
    margin-left: 30px;        
    }

    6.3 ¿Qué es box-sizing?¶

    La propiedad CSS box-sizing determina cómo se calculan las dimensiones totales de un elemento modelo de caja, incluyendo su contenido, padding y borde.

    Estas dimensiones se pueden controlar con la propiedad box-sizing, pudiendo tener dos valores: * box-sizing: content-box (Este es el valor por defecto) * box-sizing: border-box

    Por defecto, el modelo de caja (box model) se comporta como content-box. Si queremos que se comporte de la otra forma (border-box) hay que indicarlo.

    Estos dos valores hacen que el significado de las propiedades widthy height cambien.

    graph TD
    A[Modelo de caja<br> o<br> Box Model]-->B[1.- **content-box**]
    A-->C[2.- **border-box**]

    B-->|width y heigth|D(Son las medidas de **CONTENT**)

    C-->|width y height|E(Son las medidas de **BORDER**)

    6.3.1 Content-box¶

    Por defecto (comportamiento como content-box), cuando dices width: 300px, estás aplicando esta medida al contenido (párrafo, botón, imagen...). En la imagen superior en color blanco.

    Vamos a centrar el estudio en width (con heightes similar).

    El código de este model box es:

    .caja1 {
    width: 300px;           /* Ancho del área de contenido */
    height: 200px;          /* Alto del área de contenido */
    padding: 15px;          /* 15px en los 4 lados: top, right, bottom, left */
    border: 5px solid purple; /* 5px de grosor, estilo sólido, color morado */
    margin: 30px;           /* 30px de separación exterior en los 4 lados */
    background-color: lightblue; /* Color de fondo para contenido Y padding */
    box-sizing: content-box; /* width/height solo incluyen el CONTENIDO (valor por defecto) */
}

    6.3.2 La solución: border-box¶

    La solución es que se comporte como border-box, y esto se arregla con la propiedad box-sizing.

    Con box-sizing: border-box, el width incluye hasta el exterior del borde, es decir, todo el paquete (Width + 2 veces Padding + 2 veces Border):

    height: 200px;          /* alto del contenido (en lugar de 200px) */
    padding: 15px;          /* espacio interior (igual) */
    border: 5px solid purple; /* borde (igual) */
    margin: 30px;           /* espacio exterior (igual) */ 
    background-color: lightblue; /* color del contenido + padding */
    box-sizing: border-box; /* cambiamos el comportamiento */

    6.3.3 Ejemplo con elementos reales¶

    /* Todos los elementos HTML usan el mismo modelo */

p {
    padding: 10px;    /* Espacio alrededor del texto */
    border: 1px solid gray;
    margin: 15px 0;   /* Espacio entre párrafos */
}

.boton {
    padding: 15px 30px;   /* Espacio dentro del botón */
    border: 2px solid blue;
    margin: 10px;         /* Espacio alrededor del botón */
}

img {
    border: 3px solid black;
    padding: 5px;         /* Marco blanco alrededor de la imagen */
    margin: 20px;
}

    Todos estos elementos (<p>, .boton, <img>) siguen el mismo Box Model.

    6.3.4 Consejo esencial¶

    Usa esto siempre al inicio de tu CSS:

    * {
    margin: 0;
    padding: 0;
    box-sizing: border-box;
}

    Esto aplica border-box a todos los elementos HTML de tu página, facilitando los cálculos.
