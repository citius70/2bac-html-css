    Pruebas rápidas: legibilidad y contraste¶

    Esta página ofrece comprobaciones visuales rápidas para probar legibilidad y contraste en distintos anchos (simula dispositivos comunes). Sigue las instrucciones abajo para un test manual rápido.

    Instrucciones rápidas

    • Usa la barra de herramientas del navegador (DevTools → Device Toolbar) o visualiza los contenedores que hay más abajo para comprobar cómo se lee el texto a distintos anchos.
    • Para comprobar contraste numéricamente, utiliza el comprobador de WebAIM: https://webaim.org/resources/contrastchecker/
    • Revisa las recomendaciones de contraste en MDN: https://developer.mozilla.org/es/docs/Web/Accessibility/Understanding_Contrast_Ratios

    Pares de color de ejemplo¶

    • Comprobar estos pares visualmente y después pásalos al Contrast Checker.
    Mobile — 320px
    Texto: negro sobre blanco — contraste alto
    Colores: #000000 / #ffffff
    Enlace/CTA: azul sobre blanco
    Colores: #1a73e8 / #ffffff
    Texto claro sobre fondo oscuro
    Colores: #f5f5f5 / #222222
    Mobile — 375px
    Texto sobre color (ej. botón)
    Colores: #ffffff / #ffb300
    Texto gris sobre fondo blanco (contraste medio)
    Colores: #666666 / #ffffff
    Tablet — 480px
    CTA verde
    Colores: #ffffff / #0f9d58
    Desktop — 768px
    Texto principal en desktop
    Colores: #333333 / #ffffff
    Large — 1024px
    Alerta/Destructivo
    Colores: #ffffff / #ff5252

    Consejos de uso¶

    • Para cada par: copia los hex y pégalos en el Contrast Checker de WebAIM. Debes buscar un ratio >= 4.5:1 para texto normal y >= 3:1 para texto grande.
    • Comprueba también tamaños de fuente pequeños y elementos de UI con bajo contraste (ej. placeholders, labels desactivados).
    • Si usas colores semitransparentes sobre imágenes, comprueba el contraste con la composición final (puedes tomar capturas y probarlas en la herramienta).

    Verificación rápida con DevTools¶

    1. Abre DevTools (F12) y activa Device Toolbar (Ctrl+Shift+M) — selecciona un dispositivo o usa anchura personalizada.
    2. Revisa las cajas de ejemplo y confirma que los textos sean legibles a cada ancho.
    3. Usa la función de inspección para copiar valores de color si necesitas pasarlos al comprobador.

    Nota: Esta página es para pruebas manuales rápidas. Para pruebas automatizadas de contraste considera integrar herramientas como axe-core o pa11y en tu proceso de CI.
