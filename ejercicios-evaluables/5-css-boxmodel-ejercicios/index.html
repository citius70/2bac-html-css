    5. Ejercicio: “Sobre mí” y Box Model en tu Web Personal¶

    1. Actualiza tu sección “Sobre mí”, mejorando la presentación del bloque principal con Box Model:

    En tu archivo principal (index.html), dentro de la sección o <section id="sobre-mi">:

    <section id="sobre-mi">
    <h2>Sobre mí</h2>
    <div class="box-model-personal">
        <img src="foto.jpg" alt="Mi foto de perfil" title="Tu Nombre">
        <p>
            Hola, mi nombre es <strong>Tu Nombre</strong> y estudio <strong>2º de Bachillerato</strong>. 
            <em>Me apasiona la tecnología</em> y quiero aprender a crear páginas web. 
            Este bloque aplica el modelo de caja de CSS.
        </p>
    </div>
</section>
    1. Añade o modifica tu archivo de estilos externo (estilos.css), usando el modelo de caja completo y box-sizing:
    .box-model-personal {
    width: 320px;
    padding: 24px;         /* Espacio interno */
    margin: 32px auto;     /* Centrado horizontal y margen externo */
    border: 4px solid #0076ff; /* Borde visible */
    border-radius: 12px;
    background: #f0f5ff;   /* Color de fondo */
    box-sizing: border-box; /* El ancho incluye borde y padding */
    box-shadow: 0 6px 16px rgba(0,80,200,0.1); /* Sombra para resaltar el bloque */
    display: flex;
    flex-direction: column;
    align-items: center;
    gap: 16px;
}

/* Imagen adaptada como parte del modelo de caja */
.box-model-personal img {
    width: 80px;
    height: 80px;
    border-radius: 50%;            /* Foto redonda */
    border: 2px solid #0094ff;
    margin-bottom: 12px;
    object-fit: cover;
}
    1. Contexto y explicación breve (puedes incluir como comentario en el archivo o actividad): 
      <!--
    La clase box-model-personal demuestra el modelo de caja en CSS:
    Observa cómo el padding, border, margin y background afectan el tamaño y el estilo de tu bloque 'Sobre mí'.
    Cambia valores y revisa el resultado. ¿Qué sucede si cambias box-sizing a content-box?
-->

    Opciones de ampliación:
    - Añade un cuadro similar en la sección de habilidades, intereses o proyectos para comparar modelos de caja en diferentes partes de la página. - Compara el resultado visual y el espacio ocupado con/without box-sizing: border-box.
