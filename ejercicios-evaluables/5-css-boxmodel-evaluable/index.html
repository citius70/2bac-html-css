    5 css boxmodel evaluable

    Aquí tienes un ejercicio evaluable que amplía el proyecto de web personal, enfocándose en practicar el Box Model de CSS, ideal como continuación de los anteriores:

    Ejercicio Evaluable: El Box Model en tu Web Personal¶

    Objetivo:
    Aprender y aplicar el concepto de Box Model en CSS para mejorar diseño, separación y estructura visual en una página web personal ya creada en ejercicios anteriores.

    Enunciado¶

    Partiendo de tu web personal creada en ejercicios previos:

    1. Identifica y selecciona al menos 4 cajas (elementos bloque o inline-block) de tu web: pueden ser secciones, artículos, imagen de perfil, tarjeta de intereses, barra de navegación, etc.

    2. Personaliza el Box Model de cada una usando las siguientes propiedades de CSS:

    3. width y/o height
    4. margin (cuatro lados: arriba, derecha, abajo, izquierda)
    5. padding (cuatro lados: arriba, derecha, abajo, izquierda)
    6. border (tipo, color, grosor, radio)

    7. box-sizing (content-box y/o border-box)

    8. Visualiza la diferencia entre una caja con muchos márgenes y paddings, y otra sin casi espacio alrededor/interior.

    9. Añade al menos un comentario explicativo en el CSS sobre cuándo y por qué usas cada propiedad.

    10. Valida el resultado final:

    11. Las cajas deben verse separadas, ordenadas y estéticamente diferenciadas.
    12. Usa colores distintos en bordes y fondos para comprobar el efecto.
    13. Explica brevemente en un párrafo qué has aprendido sobre el Box Model.

    Ejemplo ilustrativo¶

    /* Barra de navegación con box model personalizado */
nav {
  background: #e3f2fd;
  border: 2px solid #1976d2;
  border-radius: 6px;
  padding: 10px 30px;
  margin: 20px 0 20px 0;
  box-sizing: border-box; /* Incluye el padding y border en el ancho total */
}

/* Sección principal */
main {
  background: #fffbe7;
  border: 4px dashed #ecb300;
  padding: 25px;
  margin: 40px auto 0 auto;
  box-sizing: content-box; /* El ancho NO incluye padding/border */
}

/* Perfil */
.profile-img {
  border: 5px solid #009688;
  border-radius: 50%;
  padding: 8px;
  margin: 30px 15px;
  width: 120px;
  height: 120px;
}

/* Comentario en CSS: margenes generan espacio externo, padding espacio interno */

    Checklist de entrega¶

    • Has modificado mínimo 4 elementos usando varias propiedades del box model.
    • El efecto visual de borde, separación y fondo es visible en la web.
    • Has usado y comentado box-sizing al menos en 2 elementos.
    • Entregaste el código CSS con comentarios claros.
    • Añadiste una breve reflexión sobre el aprendizaje del Box Model.
