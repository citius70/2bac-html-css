    9. Ejercicio Evaluable: Flexbox en tu Web Personal¶

    Objetivo:
    Modificar y mejorar tu web personal usando Flexbox para organizar el contenido de manera flexible y adaptable. Practicar las propiedades principales y comprobar su efecto en la disposición de las cajas.

    9.1 Enunciado¶

    En tu proyecto de web personal:

    1. Selecciona al menos dos secciones o grupos de elementos que quieras organizar: ejemplos comunes son la barra de navegación, una galería de intereses, tarjetas de perfil, lista de proyectos, etc.

    2. Modifica su CSS para convertirlos en contenedores flexibles:

    3. Añade display: flex; y explora distintas direcciones (flex-direction: row, column).

    4. Usa flex-wrap: wrap; en al menos uno para permitir múltiples líneas si hay muchos elementos.

    5. Prueba y documenta tres combinaciones diferentes de las siguientes propiedades en tus contenedores:

    6. justify-content (ejemplo: center, space-between)
    7. align-items (ejemplo: flex-start, center, flex-end)

    8. Cambia la orientación con flex-direction

    9. En al menos dos elementos hijos, experimenta con:

    10. flex-grow: haz que una caja ocupe más espacio que las demás.
    11. flex-shrink: limita cómo se encoge si falta espacio.
    12. flex-basis: define tamaño base.
    13. order: cambia el orden visual de aparición respecto al HTML.

    14. align-self: alinea individualmente una caja en el eje secundario.

    15. Incluye comentarios en el CSS explicativos y un pequeño párrafo en tu web donde reflexionas qué ventajas ofrece Flexbox y qué dificultades encontraste.

    16. Visualiza tu resultado:

    17. Ajusta el ancho de la ventana para demostrar la adaptabilidad.
    18. Usa colores, bordes y distintos tamaños para ayudar a identificar el efecto de cada propiedad.

    9.2 Ejemplo ilustrativo de entrega¶

    /* Barra de navegación como contenedor flexible horizontal */
nav {
  display: flex;
  flex-direction: row;
  justify-content: space-between;
  align-items: center;
  background: #e3f2fd;
  padding: 10px 30px;
}

/* Galería de intereses con varias líneas */
.galeria-intereses {
  display: flex;
  flex-wrap: wrap;
  justify-content: center;
  gap: 14px;
}

.galeria-intereses .item {
  flex-basis: 120px;
  flex-grow: 1;
  background: #fcf3cf;
  padding: 12px;
  margin: 6px;
  border-radius: 8px;
  border: 2px solid #f7c948;
}

/* Elemento destacado: ocupa más espacio y cambia orden visual */
.galeria-intereses .item.destacado {
  flex-grow: 3;
  order: -1;
  background: #aed9e0;
}

/* Botón de perfil alineado al fondo del grupo */
.galeria-intereses .item.perfil {
  align-self: flex-end;
  background: #b6fcd5;
}

/* Comentarios explicativos en CSS */

    Reflexión:
    Con Flexbox, la adaptabilidad y el orden de los elementos mejoran mucho. Es fácil centrar, repartir espacio y cambiar el orden visual. La mayor dificultad la encontré al querer combinar flex-basis y flex-grow, y entender cómo funcionan juntos al cambiar el ancho de la página.

    9.3 Checklist de entrega¶

    • Hay al menos dos contenedores con Flexbox y distintas configuraciones.
    • Se usan varias propiedades de Flexbox en los hijos.
    • El efecto visual es claro y visible (usa colores, bordes, tamaños diferentes).
    • Se han añadido comentarios y una breve reflexión escrita.
    • Se prueba la web en varios tamaños para comprobar la flexibilidad.
