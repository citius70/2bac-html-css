    8. Ejercicio Evaluable: Entendiendo position en tu Web Personal¶

    Objetivo:
    Aplicar y experimentar con las distintas formas de posicionar elementos en CSS para comprender cómo se comportan y cuándo utilizarlas en la web.

    8.1 Enunciado¶

    1. Selecciona una copia de tu web personal (o una versión de prueba).
    2. Busca o crea al menos tres elementos que puedas reposicionar en tu web, por ejemplo:
    3. Una barra de menú
    4. Un botón flotante de ayuda o volver arriba
    5. Un badge/decoración/aviso sobre una imagen
    6. Un encabezado o mensaje destacado
    7. Aplica a cada uno un valor diferente de la propiedad position:
    8. Uno con relative
    9. Uno con absolute
    10. Uno con fixed
    11. (opcional: uno con sticky si tu web tiene scroll suficiente)
    12. En cada caso, utiliza alguna combinación de las propiedades top, right, bottom y left para que el cambio de posición sea MUY VISIBLE en la pantalla.
    13. Personaliza visibilidad: pon color de fondo, borde y un tamaño llamativo para que se aprecie el efecto.
    14. Incluye comentarios en el CSS explicando el porqué de cada elección y cómo afecta al resto.
    15. Agrega una pequeña reflexión al final de tu entrega sobre qué valor de position ves más útil y en qué situaciones lo usarías.

    8.2 Ejemplo ilustrativo¶

    /* Menú fijo en la parte superior */
.menu-fijo {
  position: fixed;
  top: 0;
  left: 0;
  width: 100%;
  background: #ffecb3;
  border-bottom: 2px solid #ff8f00;
  padding: 10px 0;
  z-index: 999;
  text-align: center;
  /* Uso fixed para que el menú sea visible aunque hagas scroll */
}

/* Botón flotante en la esquina inferior derecha */
.boton-flotante {
  position: fixed;
  right: 20px;
  bottom: 20px;
  background: #81d4fa;
  border: 2px solid #0288d1;
  border-radius: 50%;
  width: 60px;
  height: 60px;
  font-size: 2em;
  color: #01579b;
  text-align: center;
  box-shadow: 2px 2px 8px #888;
  /* Uso fixed: siempre a la vista en la ventana */
}

/* Badge decorativo absoluto sobre foto */
.badge-abs {
  position: absolute;
  top: 20px;
  left: 100px;
  padding: 6px 16px;
  background: #ffe082;
  border-radius: 16px;
  border: 2px solid #bfa500;
  font-weight: bold;
  /* Uso absolute: respecto al contenedor con position: relative */
}

/* Imagen con contenedor posicionador */
.img-contenedor {
  position: relative;
  width: 200px;
  display: inline-block;
}

/* Comentario: relative no mueve la caja, pero sí permite que "absolute" se posicione respecto a ella. */

    <div class="menu-fijo">MENÚ SUPERIOR FIJO</div>

<main style="margin-top: 60px;">
  <h2>Ejemplo de position</h2>
  <div class="img-contenedor">
    <img src="https://randomuser.me/api/portraits/men/11.jpg" width="200" alt="Foto de perfil" />
    <span class="badge-abs">VIP</span>
  </div>
  <p>Contenido de ejemplo tras el menú...</p>
  <button class="boton-flotante">?</button>
</main>

    8.3 Checklist de entrega¶

    • Has utilizado al menos tres valores distintos de position en tu web.
    • El efecto visual de cada posición es fácil de ver.
    • Hay colores, bordes o tamaños llamativos para destacar los elementos reposicionados.
    • Incluiste comentarios CSS explicativos.
    • Reflexionaste sobre el valor de position que te parece más útil.
