    6. Ejercicio Evaluable: Comparando Valores de display en la Web Personal¶

    Objetivo:
    Dominar el uso de los distintos valores de la propiedad display para controlar el comportamiento visual de los elementos en la web.

    6.1 Enunciado¶

    1. Selecciona tres grupos de elementos de tu web personal (por ejemplo: menú de navegación, lista de intereses, tarjetas de perfil/descripción).
    2. Para cada grupo, realiza pruebas con los siguientes valores de la propiedad display:
    3. block
    4. inline
    5. inline-block
    6. Modifica el CSS para que cada grupo de elementos use uno de los valores de display. Debes probar cómo:
    7. Cambia la disposición (horizontal/vertical)
    8. Cambia el respeto de ancho (width), alto (height) y espacio (margin, padding)
    9. Observa los efectos visuales en la página y muestra capturas de pantalla (o describe el resultado en tu entrega).
    10. Escribe breves comentarios en tu código explicando cuándo y por qué usarías cada valor en un proyecto web real.
    11. Incluye una reflexión final (1-2 frases) sobre cuál valor te parece más relevante y por qué para organizar el contenido de una página.

    6.2 Ejemplo breve de entregable¶

    .menu {
  display: block;   /* Ocupa toda la línea, cada elemento debajo del anterior */
}
.intereses span {
  display: inline;  /* Todos seguidos horizontalmente, sin ancho propio */
}
.card {
  display: inline-block; /* Se alinean en fila si hay espacio, puedes definir ancho/alto */
  width: 180px;
  height: 110px;
  margin: 10px;
  padding: 8px;
  border: 2px solid orange;
}

    Comentario en el CSS:

    /* Uso block para el menú porque así es más fácil separarlo visualmente y organizar las opciones en vertical */
/* Uso inline para intereses porque es contenido breve y queda más limpio en línea */
/* Uso inline-block en card porque quiero separar varias tarjetas en horizontal y definir tamaños */

    6.3 Checklist para entrega¶

    • Has aplicado block, inline y inline-block en elementos distintos.
    • Los efectos visuales son claramente distintos y explicados en tu reflexión.
    • Incluiste comentarios CSS justificando cada elección.
    • El código es claro y organizado.

    Aquí tienes un ejemplo en HTML y CSS completo para el ejercicio sobre la propiedad display en tu web personal:

    <!DOCTYPE html>
<html lang="es">
<head>
    <meta charset="UTF-8">
    <title>Ejercicio evaluable: Display en CSS</title>
    <style>
        .menu-item {
            display: block;           /* Cada opción en línea diferente */
            background: #f8bbd0;
            margin: 8px 0;
            padding: 6px;
            border-radius: 4px;
        }
        .intereses {
            margin: 20px 0;
        }
        .intereses span {
            display: inline;          /* Todos juntos, fluye como texto */
            background: #b2dfdb;
            margin-right: 10px;
            padding: 4px 9px;
            border-radius: 4px;
        }
        .card {
            display: inline-block;    /* Se alinean en horizontal, puedes ajustar tamaño */
            background: #fff9c4;
            border: 2px solid #fbc02d;
            width: 140px;
            height: 90px;
            padding: 8px;
            margin: 9px;
            border-radius: 6px;
            vertical-align: top;
        }

        /* Comentario: block para menú porque se separa mejor, inline para intereses, inline-block para mostrar tarjetas alineadas y definir tamaño */
    </style>
</head>
<body>
    <h2>Menú de Navegación (display: block)</h2>
    <div class="menu-item">Inicio</div>
    <div class="menu-item">Sobre mí</div>
    <div class="menu-item">Contacto</div>

    <h2>Intereses (display: inline)</h2>
    <div class="intereses">
        <span>Tecnología</span>
        <span>Fotografía</span>
        <span>Literatura</span>
        <span>Música</span>
    </div>

    <h2>Tarjetas de Perfil (display: inline-block)</h2>
    <div class="card">
        <strong>Nombre:</strong> Ana<br>
        <strong>Edad:</strong> 17
    </div>
    <div class="card">
        <strong>Nombre:</strong> Juan<br>
        <strong>Edad:</strong> 18
    </div>
    <div class="card">
        <strong>Nombre:</strong> Eva<br>
        <strong>Edad:</strong> 17
    </div>

    <p>
        <em>Reflexión:</em> <br>
        He usado <b>display: block</b> para crear menús verticales claros, <b>display: inline</b> para mostrar los intereses como texto fluido y <b>display: inline-block</b> para las tarjetas, que así pueden alinearse horizontalmente y fijar tamaño.
    </p>
</body>
</html>
